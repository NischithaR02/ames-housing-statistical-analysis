# Ames Housing Statistical Analysis

## Project Overview

This project explores the Ames Housing dataset to identify factors influencing residential property prices using statistical analysis and modelling techniques in R.

The work was completed as part of the MSc Data Science and Analytics programme at Cardiff University.

---

## Objectives

- Perform exploratory data analysis (EDA)
- Investigate housing price distributions
- Conduct hypothesis testing on housing features
- Analyse relationships between variables
- Develop and evaluate regression models for house price prediction

---

## Methods

### Exploratory Data Analysis
- Data cleaning and preparation
- Summary statistics
- Distribution analysis
- Correlation exploration

### Statistical Analysis
- Hypothesis testing
- Distribution fitting
- Regression modelling
- Model diagnostics and validation

---

## Tools & Technologies

- R
- ggplot2
- dplyr
- Statistical Modelling
- Linear Regression

## Key Findings

- Property size was strongly associated with sale price.
- Neighbourhood characteristics influenced housing values.
- Overall property condition showed a positive relationship with sale price.
- Regression analysis identified important predictors of residential property prices.
